Time Manager

Time Manager filters your layers and displays only layers and features that match the specified time frame. Time Manager supports vector layers and raster layers (including WMS-T).

Timestamps have to be in one of the following formats:

  • %Y-%m-%d %H:%M:%S.%f
  • %Y-%m-%d %H:%M:%S
  • %Y-%m-%d %H:%M
  • %Y-%m-%dT%H:%M:%S
  • %Y-%m-%dT%H:%M:%SZ
  • %Y-%m-%dT%H:%M
  • %Y-%m-%dT%H:%MZ
  • %Y-%m-%d
  • %Y/%m/%d %H:%M:%S.%f
  • %Y/%m/%d %H:%M:%S
  • %Y/%m/%d %H:%M
  • %Y/%m/%d
  • %H:%M:%S
  • %H:%M:%S.%f
  • %Y.%m.%d %H:%M:%S.%f
  • %Y.%m.%d %H:%M:%S
  • %Y.%m.%d %H:%M
  • %Y.%m.%d
  • %Y%m%d%H%M%SED
  • Integer timestamp in seconds after or before the epoch (1970-1-1)

The layer list contains all layers managed by Time Manager. To add a vector layer, press [Add layer]. To add a raster layer, press [Add raster] If you want to remove a layer from the list, select it and press [Remove layer].

Below the layer list, you'll find the following animation options:

Show frame for x milliseconds ... allows you to adjust for how long a frame will be visible during the animation

Play animation backwards ... if checked, the animation will run in reverse direction

Display frame start time on map ... if checked, displays the start time of the visible frame in the lower right corner of the map

Add Layer dialog

Here, you are asked to select the layer that should be added and specify the columns containing start and (optionally) end time.

The offset option allows you to further time the appearance of features. If you specify an offset of -1, the features will appear one second later than they would by default.

Dock widget

The dock was designed to attach to the bottom of the QGIS main window. It offers the following tools:

  • ... On/Off switch, allows you to turn Time Manager's functionality on/off with the click of only one button
  • [Settings] ... opens the Settings dialog where you can manage your spatio-temporal layers
  • [Export Video] ... exports an image series based on current settings (This button is only enabled if there are layers registered in Time Manager "Settings")
  • Time frame start: [2000-01-01 00:00:00] ... shows the start time of the currently active frame. Allows you to precisely specify your desired analysis time.
  • Time frame size: [x] ... allow you to choose the size of the time frame
  • ... go to the previous time frame
  • ... go to the next time frame
  • Slider ... shows the position of current frame relative to the whole dataset and allows you to seamlessly scroll through the dataset
  • ... start an automatic animation based on your current settings
